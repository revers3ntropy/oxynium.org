Oxynium

Oxynium is a statically typed, low level, compiled programming language

Key features include:

  • Powerful type system with generics
  • First-class functions and lambda functions
  • Macros
  • Classes
  • Operator overloading
  • Interoperability with other languages
  • Explicit error handling

Getting Started

Read the docs at oxynium.org/docs

Install for Linux / MacOS

curl -sSL https://oxynium.org/scripts/install | bash

start a new terminal session, and then

oxy hello_world.oxy

Examples 

print("Hello, world!")
const N = 10
def fibonacci (n: Int) Int {
    if n <= 1 ->
        return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

print(fibonacci(N).Str()); // 55
def main() {
    for i in range(100) {
        print(fizzbuzz(i));
    }
}

def fizzbuzz (n: Int) Str {
    if n % 3 == 0 \&\& n % 5 == 0 {
        return "FizzBuzz";
    }
    if n % 3 == 0 -> return "Fizz";
    if n % 5 == 0 -> return "Buzz";
    return n.Str();
}
